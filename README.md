# Flavor Fusion 🍛

**Flavor Fusion** is a dynamic and visually rich web-based application showcasing authentic Indian cuisine. Built using **React (via Babel)** and styled with **CSS**, it offers a seamless, responsive UI for browsing, searching, and adding dishes to a cart — simulating an online food ordering experience.

## 🚀 Features

- 🍽️ Interactive food menu with categories like Veg, Non-Veg, Bread, and Desserts
- 🔍 Live search functionality
- 🛒 Cart system with quantity adjustment and total price calculation
- 📱 Fully responsive design for mobile and desktop
- 💬 Modal-based login interface (placeholder for future API integration)
- 🎨 Clean and modern design using CSS variables and Google Fonts
- 📸 Image-rich presentation with Unsplash and custom food images

## 🛠️ Built With

- HTML5 + CSS3
- [React (via CDN)](https://reactjs.org/)
- [Babel Standalone](https://babeljs.io/)
- [Font Awesome](https://fontawesome.com/)
- [Google Fonts - Poppins](https://fonts.google.com/specimen/Poppins)
